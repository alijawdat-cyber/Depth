### المواصفات التقنية - الأمان
- **التحقق بخطوتين**: إلزامي لجميع الأدمنز
- **محاولات الدخول**: 3 محاولات قبل القفل
- **قفل الحساب**: 30 دقيقة بعد 3 محاولات فاشلة
- **Audit Log**: تسجيل جميع محاولات الدخول
