🎯 مثال مكوّن HeaderSidebar

هذا مثال تطبيقي لمكوّن الهيدر والسايدبار المدموج الذي يدعم العربية والإنجليزية.

✨ المميزات الرئيسية

🎚️ هيدر ثابت متجاوب

يبقى في الأعلى مع تأثيرات التمرير وخلفية شفافة.

🍞 فتات خبز بالإنجليزية

تنقل واضح يظهر مسار الصفحة الحالية.

📱 تصميم متجاوب

يعمل بشكل مثالي على جميع أحجام الشاشات.

🌓 وضع فاتح/داكن

دعم كامل للوضعين الفاتح والداكن.

♿ الوصولية

مصمم ليكون قابل للوصول لجميع المستخدمين.

🔗 تكامل GitHub

رابط مباشر للمستودع على GitHub.

🔧 كيفية الاستخدام

يمكنك استخدام هذا المكوّن في مشروع React/Next.js كالتالي:

import HeaderSidebar from './HeaderSidebar';

const navigation = [
  {
    id: 'home',
    label: 'الرئيسية',
    href: '/',
    children: []
  },
  // ... المزيد من العناصر
];

function App() {
  return (
    <HeaderSidebar 
      navigation={navigation}
      githubUrl="https://github.com/your-repo"
    />
  );
}

📝 ملاحظات التطوير

  • الاتجاه: الهيدر يدعم RTL للعربية، فتات الخبز LTR للإنجليزية
  • الوصولية: جميع العناصر تدعم قارئات الشاشة
  • الأداء: استخدام CSS transforms للانيميشن
  • المرونة: قابل للتخصيص بسهولة عبر CSS Variables
